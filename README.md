# Dependencies

- Node 10.x or above
- NPM 5.x or above
- PG 6.x
- Dotenv 2.0.0 or above
- Express 4.17.1 or above
- Mailgun 0.5.0 or above
- Uuid 9.0.1 or above
- Bcrypt 5.1.1 or above
- Body-parser 1.20.2 or above
- Moment 2.30.1 or above
